Download the executable jar file from github

Open up the command prompt by clicking the search bar in
bottom right and searching command prompt

You need to make sure that the command prompt is able to
find the jar file you just download

To do this
 - Type in the command prompt "cd " without the quotes
 - Find where you downloaded the jar file
 - Drag the file into the command prompt
 - Delete off the end of what pops up "\UMLEditor.jar"
	without the quotes
 - Make sure you do not delete the ending "

Then click enter

Lastly type in "java -jar UMLEditor.jar" without the quotes

The program should be running now, enjoy!
